## VSCode好用插件介绍

### LiveCode for python

```python
LiveCode evaluates your python code while you type and displays variable values for each line.	
当你为每行输入和显示变量值时，LiveCode会评估你的python代码。  

Open a python file and click on icon in the top bar to the right to open LiveCode. Click on the icon again to close LiveCode.
打开一个python文件，点击右上角的图标打开LiveCode。 再次单击图标关闭LiveCode。 

Or run LiveCode through the command search: `control-shift-p`
or use the shortcuts: `control-shift-a` (current doc) / `control-shift-q` (new doc)
或者通过命令搜索运行LiveCode: ' control-shift-p '  
或者使用快捷键:' control-shift-a '(当前文档)/ ' control-shift-q '(新文档)再次单击图标关闭LiveCode。
```

### Python Docstring Generator

```python
Automatically generates detailed docstrings for python functions
自动为python函数生成详细的文档字符串

Usage
Cursor must be on the line directly below the definition to generate full auto-populated docstring

Press enter after opening docstring with triple quotes (三引号)
Keyboard shortcut: ctrl+shift+2
Can be changed in Preferences -> Keyboard Shortcuts -> extension.generateDocstring
Command: Generate Docstring
Right click menu: Generate Docstring

使用
光标必须位于函数定义的正下方，才能生成自动填充的完整文档字符串
在用三重引号()打开文档字符串后按回车键
键盘快捷键:ctrl+shift+2
可以在Preferences -> Keyboard Shortcuts -> extension.generateDocstring
命令:生成文档字符串
右击菜单:生成文档字符串
```
